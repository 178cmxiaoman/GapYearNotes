# GitHub学习笔记
### 初始化操作
首先要在github上创建好仓库
```
echo "name" >> README.md
git init
git add README.md
git commit -m "备注"
git branch -M main
git remote add origin git@github.com:178cmxiaoman/GapYearNotes.git
git remote add git push -u origin main
```

### 对于以及初始化过的仓库
```
git remote add origin git@github.com/....git
git branch -M main
git push -u orgin main
```
